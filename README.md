服务网站查缺查重 (cqcc)
================
大萌	2022

　　网站是有关部门信息公开的法定渠道之一，但由于各种原因，某些服务网站存在缺失、重复的情况，本项目旨在记录这些缺失或重复的网站，以利于我国服务网站的建设。

410100	郑州市辖区公安机关缺少微博	2022/7/20

　　河南省郑州市只有市公安局有微博账号：[平安郑州](https://weibo.com/zzgawb)，下辖六区五市一县的公安局均搜不到微博。与之对比的是，开封市下辖五区四县的公安局有七个开通了微博。全国县级公安机关大部分都开通了微博账号。
